# seasonal
### 仿果酷网
功能流程：
    用户浏览商品，可对用户浏览行为做记录，
    用户收藏商品、用户把商品加入购物车。
    用户进入购物车，选择要支付的商品。
    点击支付后生成订单：
    1.未支付：订单生成在个人中心可后续支付。
    2.确认支付：生成总价与附近商家的地图，用户可选择自取商家，为用户分配自取密码与账号。用户选择地址配送，会根据距离自动生成一个配送费。
   确认付款后，弹出支付宝支付界面，可以选择扫码，或登录支付宝支付。
   支付完成后给出用户提示。
   支付完成后在个人中心的评论页面中有未评论商品，可供用户评论商品。
   个人中心有地址管理个人信息管理。
